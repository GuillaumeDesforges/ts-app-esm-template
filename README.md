# `ts-app-esm-template`

A template for a web app written in TypeScript using ESM

> This is still a WIP

## Setup

Requirements:

- yarn
- NodeJS >= 18.x

```
yarn install
```

## Build

```
yarn build
```

in development, you can use the `--watch` mode of `tsc`:

```
yarn build:watch
```

## API

Run the API:

```
node api
```

## Client

Run the development server (requires the API to be running):

```
yarn client:dev
```
