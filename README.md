# `ts-app-esm-template`

A template for a web app written in TypeScript using ESM

> This is still a WIP

## Setup

Requirements:
* yarn
* NodeJS >= 18.x

```
yarn install
```

## Build

```
yarn build
```

in development, you can use the `--watch` mode of `tsc`:

```
yarn build:watch
```

## Run

```
node api
```
